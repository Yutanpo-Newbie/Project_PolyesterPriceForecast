# Project_PolyesterPriceForecast
大学の実習科目にて
ベトナムに衣服製造拠点を持つアパレル企業をターゲットに、ポリエステルの仕入れのタイミングを検討する機会を与える指標の開発。
「原料価格と世界の衣服生産需要を考慮したポリエステル仕入れ価格予測モデル」

概要：
ベトナムの衣服製造拠点から見た場合のポリエステル仕入れ価格を予測するモデルの開発のために以下の観点から開発を行っている。
・先行事例「Price prediction of polyester yarn based on multiple linear regression model」の論文では、このポリエステル価格を予測するうえで、為替、原料価格および工場稼働率を変数として入力した際に精度が高まることが検証されている。これに従って、為替、ポリエステル原料のBrent（原油価格）、PTA&MEG（原油加工品価格）の原料価格の変数を入力するだけでなく、独自にアパレル市場の需要を表現するものとして、アジア地域の主要衣服生産国の衣服輸出総額の合計、綿花価格変数に加える。

モデル：単一ステージの多変量時系列回帰アルゴリズム
アルゴリズム：XGBoost
データ単位：2016/01～2023/12の月次
評価方法：RMSE、MAPE
